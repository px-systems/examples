# px-systems/examples

End-to-end walkthroughs for [PX](https://px.app), the parallel execution CLI.

Each subdirectory is a self-contained project: code, `px.yaml`, and a README that walks through running it locally and on a cluster.
New to PX?
Start with the [Quick Start](https://px.app/docs/quick-start), then come back here for runnable scenarios.

## Walkthroughs

- [`python-sha256sum-images`](python-sha256sum-images/): hash NASA Artemis II images in parallel, locally then on a cluster.
  Stdlib Python, zero deps.
